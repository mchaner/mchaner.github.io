# Los Métodos

En la programación, un método es una subrutina cuyo código es definido en una clase y puede pertenecer tanto a una clase, como es el caso de los métodos de clase o estáticos, como a un objeto, como es el caso de los métodos de instancia.

Estructura de un método:

/* static => modificador
     * void => tipo o retorno
     * Metod => nombre del método
     * () => donde enviaremos los parámetros
     */
    static void Method(int num, string texto)
    {
        //instrucciones de código
    }

    //Método sin parámetros, ni tipo
    static void Method()
    {
        //instrucciones de código
    }

    //Método sin parámetros, pero con tipo
    static int Method()
    {
        int num = 0;
        return num;
    }

    //Método con parámetros, pero sin tipo
    static void Method(int num)
    {
        Console.Writeline(num);        
    }

    //Método con parámetros y con tipo
    static int Method(int num)
    {
        int response = num;
        return response;        
    }
