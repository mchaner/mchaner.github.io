# Introducción a C# y .NET

# Que es C#

C# es un lenguaje de programación desarrollado por Microsoft, orientado a objetos, que ha sido diseñado para compilar diversas aplicaciones que se ejecutan en .NET Framework. Se trata de un lenguaje simple, eficaz y con seguridad de tipos . Las numerosas innovaciones de C# permiten desarrollar aplicaciones rápidamente y mantener la expresividad y elegancia de los lenguajes de estilo de C.

La sintaxis viene derivada de C y C+ + y utiliza el modelo de objetos de la plataforma .NET, muy parecido al de Java, aunque incluye mejoras propias de otros lenguajes .

Curiosidad

Como curiosidad, el nombre de este lenguaje fue inspirado por la escala musical. En ella, la letra C equivale a la nota musical do y el símbolo # significa sostenido, lo que indica que es un semitono más alta. Así, C# sugiere que es superior a C y C++.

# .NET Framework

Un framework es un patrón o esquema que ayuda a la programación a estructurar el código y a ahorrar tiempo y esfuerzos a los programadores, en este caso esta creada por Microsoft, con la cual podemos crear aplicaciones para distintos entornos (PC, Móviles, WEB).

Este framework puede ser utilizado desde dos perspectivas, como usuario y como desarrollador. Como usuario solo debemos tenerlo instalado y nuestros programas se ejecutaran sin problemas, si somos detallistas nos habremos dado cuenta que Windows instala este framework por defecto. Y como Desarrollador debemos instalar la version que satisfaga nuestras necesidades, ya que cada version nueva aporta funcionalidades que antes no estaban.

.NET, admite mas de 60 lenguajes de programación, pero los mas comunes o conocidos son: C#, C++, VB, F# entre otros.

# Componentes principales de .NET

CLR: Common Language Runtime, es el encargado de leer el código y ejecutarlo.

FCL: Framework Class Library, contiene códigos y datos ya pre escritos, para ser usados por otros.

# Para crear una aplicación .NET

Para poder crear nuestra primera aplicación necesitaremos de un compilador, y en este caso existen dos conocidos, uno es Visual Studio y el otro MonoDevelop, nosotros trabajaremos con el primero. Visual Studio es un IDE (Entorno de desarrollo integrado), por lo cual posee un compilador, un depurador y un constructor de interfaces gráficas.

TIP

Compilador: es el encargado de traducir nuestro código de alto nivel al lenguaje maquina, para que nuestro ordenador pueda ejecutarlo.

Descarga

Visual Studio => Descargar Visual Studio (opens new window)

  • Al instalar nuestro IDE, debemos seleccionar las cargas de trabajo que utilizaremos, como se muestra a continuación:

Este proceso tomara un tiempo, dependiendo de la velocidad de nuestra conexión a internet.

  • Una vez descargado e instalado nuestro IDE, lo ejecutaremos y veremos la siguiente pantalla, y seleccionamos Crear un proyecto

  • Después seleccionamos un proyecto de tipo Aplicación de consola(.NET Framework), y que sea en lenguaje c#. La próxima vez que seleccionemos el tipo nos aparecerá en plantillas de proyecto recientes.

  • Acto seguido, configuramos nuestro proyecto:

TIP

  • Nombre del proyecto: es el nombre que le asignaremos al proyecto en concreto, ya que una solución puede tener mas de un proyecto.
  • Ubicación: es donde se creara y guardara nuestra solución
  • Nombre de la solución: es el nombre que tendrá la solución, esta puede tener uno o mas proyectos, por defecto colocara el mismo nombre del proyecto, pero esto puede cambiarse sin ningún problema.
  • Framework: debemos seleccionar la version de Framework con la que trabajaremos, atendiendo a las necesidades que tengamos.

# El ambiente de desarrollo

  • Ahora revisemos el area de trabajo

    • Bueno en la parte superior tenemos la barra de herramientas, como son demasiadas opciones los invito a revisarlas con calma, pero nos centraremos ahora particularmente en el menu compilar => compilar solución y en el menu Depurar => Iniciar sin depurar.

    • Al costado izquierdo, vemos dos pestañas verticales, "Explorador de servidores" en el cual podemos gestionar nuestras conexiones a otros entornos como por ejemplo la base de datos y "Cuadro de herramientas", en el cual veremos los elementos para crear interfaces gráficas cuando tenemos un proyecto de este tipo, como ahora es de consola nos aparece vació.

    • Al centro tenemos el editor de código, es en donde escribiremos nuestras lineas de código y crearemos nuestros algoritmos.

    • Al costado derecho tenemos el "Explorador de soluciones", y como su nombre lo indica podremos navegar por nuestra solución y revisar nuestro o nuestros proyectos. También en este costado pero en la parte inferior veremos una ventana "Propiedades" esta ventana nos mostrara las propiedades de algún componente seleccionado en nuestra interfaz gráfica, pero como nuestro proyecto es de consola no veremos nada.

    • Por ultimo, en la parte inferior, hay una ventana de "Salida", que es en donde fe mostrara información de la compilación de nuestro proyecto.

# Compilar y ejecutar

Para compilar nuestro proyecto, nos vamos al menu compilar => Compilar solución, después de ver que la compilación fue correcta podemos ejecutar la aplicación con depuración o sin depuración, en nuestro caso seleccionaremos Depurar => Iniciar sin depurar.

TIP

Cuando ejecutamos nuestra aplicación con depuración, la consola se cerrara inmediatamente después de ejecutarse la aplicación, para evitar eso deben agregar al final del código la siguiente linea:











 




using System;

namespace AplicacionConsola
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hola Mundo");
            //Esta linea evitara el cierre de la consola
            Console.ReadKey();
        }
    }
}
