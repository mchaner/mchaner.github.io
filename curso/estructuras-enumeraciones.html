# Que son las estructuras

Son tipos de datos definidos por el programador, estos tipos de datos contienen otros tipos de datos definidos por el lenguaje a los cuales llamaremos campos.

Si tomamos como ejemplo una agenda, la estructura se llamaría agenda:

Si llevamos esto a código quedaría de la siguiente manera:

//esto debe estar fuera del main
public struct Agenda
{
    public string Nombre;
    public string Direccion;
    public string Telefono;
    public int Edad;
}
