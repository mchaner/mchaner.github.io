# Las Colecciones

# El ArrayList

Es una lista de datos, en especifico el ArrayList es una clase, la cual posee sus propios métodos, con los cuales podemos añadir datos en la última posición con "Add", insertar en cualquier otra con "Insert", recuperar cualquier elemento usando corchetes, o incluso ordenar toda la lista con "Sort".

# Uso del Arraylist

Con esto instanciamos y declaramos el ArrayList.

//Si sabemos el tamaño del arraylist podemos asignarlo entre los paréntesis
ArrayList lista = new ArrayList(5);
1
2

Para poder añadir elementos al ArrayList podemos utilizar su método Add, el cual añadirá lo que necesitemos al ultimo lugar de la lista

//Añadimos un entero a la cola de lista
lista.Add(5);
lista.Add(8);
lista.Add(15);
1
2
3
4

Para poder acceder a los datos del ArrayList debemos usar el Indice al igual que un Array

Console.WriteLine(lista[0]);
1

Ademas del Add podemos realizar una inserción, que nos permite insertar un elemento en alguna posición deseada usando el indice

//Necesitamos el indice y el valor que deseamos insertar
lista.Insert(1,20);
1
2

Para remover un elemento solo necesitamos el indice

lista.RemoveAt(1);
1

Para buscar un elemento en el ArrayList, podemos utilizar el método IndexOf, para ello necesitamos entregarle el valor y este nos retorna el indice del ArrayList

int indice = lista.IndexOf(15);
Console.WriteLine(indice);
1
2

También podemos recorrer el ArrayList, con Foreach

foreach (int item in lista)
{
    Console.WriteLine(item);
}

# El Hashtable

Es una colección que permite almacenar pares de objetos, donde el primero es conocido como llave (key) y el segundo es conocido como valor (value)

# Uso del Hashtable

Para instanciar y declarar un hashtable

Hashtable table = new Hashtable();
1

Para agregar elementos debemos considerar dos objetos el key y el value

table.Add("Pedro", 23);
table.Add("Luis", 25);
table.Add("Diego", 21);
1
2
3

Para recorrer un hashtable podemos usar la estructura DictionaryEntry de la siguiente manera

foreach (DictionaryEntry item in table)
{
    Console.WriteLine("La key es {0} y el value es {1}", item.Key, item.Value);
}
1
2
3
4

Limpiar el Hashtable

table.Clear()
1

El Hashtable al trabajar con una key esta no puede repetirse por lo que deberíamos validar si esta key existe antes de asignarla

bool isExist;
isExist = table.Contains("Luis");
1
2

Ahora si necesitamos encontrar el valor en el Hashtable hacemos lo siguiente

table.ContainsValue(25);
1

Para remover un elemento del Hashtable necesitamos solo la key

table.Remove("Luis");
1
