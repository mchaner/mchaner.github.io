# Estructuras de control

Las estructuras de control, son instrucciones que permiten romper la secuencia de la ejecución de un programa; esto significa que una estructura de control permite que se realicen unas instrucciones y omitir otras, de acuerdo a la evaluación de una condición.

# IF

Tanto en la vida real como en la programación nos veremos enfrentados a tomar una decision para ver que camino debe seguir cierto evento, esto quiere decir que algo deberá suceder si se cumple cierta condición o también cuando no se cumpla.

Para resolver este tipo de decisiones necesitamos conocer los operadores de igualdad y de comparación los cuales se evaluaran y deberán entregar un resultado de condición boleana (true o false).

using System;

namespace AplicacionConsola
{    
    class Program
    {
        static void Main(string[] args)
        {
            int a = 4;
            int b = 6;

            // False
            Console.WriteLine(a == b);
            //True
            Console.WriteLine(a != b);

            int c = 7;
            int d = 7;

            //True
            Console.WriteLine(c == d);
            //False
            Console.WriteLine(c != d);

            Console.ReadKey();
        }
    }
}

Expresión ternaria

Este es otro tipo de if en el cual podemos evaluar sentencias sencillas y suele ser muy util.

# SWITCH

A diferencia del if (que evaluá una condicion), el switch compara una condicion con múltiples casos (expresión de coincidencia). Entendiendo que es similar a un if si tenemos mas de dos opciones condicionales se recomienda usar switch.

using System;

namespace AplicacionConsola
{
    class Program
    {
        static void Main(string[] args)
        {
            int num = 2;
            string numero;
            switch (num)
            {
                case 1:
                    numero = "uno";
                    break;
                case 2:
                    numero = "dos";
                    break;
                case 3:
                    numero = "tres";
                    break;
                case 4:
                    numero = "cuatro";
                    break;
                case 5:
                    numero = "cinco";
                    break;
                default:
                    numero = "error";
                    break;
            }

            Console.WriteLine(numero);

            Console.ReadKey();
        }

    }
}

# Los Ciclos, Iteraciones.

Son instrucciones para repetir procesos ordenadamente, según una secuencia lógica, bien definida y finita.

# Ciclo FOR

En resumen, un ciclo for es una estructura de control iterativa, que nos permite ejecutar de manera repetitiva un bloque de instrucciones, conociendo previamente un valor de inicio, un tamaño de paso y un valor final para el ciclo.

using System;

namespace AplicacionConsola
{
    class Program
    {
        static void Main(string[] args)
        {
            // "int i = 0" esto es el valor inicial
            // "i < 10" esto es el Valor Final
            // "i++" esto es el tamaño del paso
            for (int i = 0; i < 10; i++)
            {
                Console.WriteLine(i);
            }

            Console.ReadKey();
        }
    }
}

# Ciclo FOREACH

La estructura repetitiva foreach es utilizada para recorrer colecciones de datos (por ejemplo listas de datos), si bien podemos utilizar de forma tradicional las otras estructuras repetitivas el empleo del foreach hacen más natural el acceso a los elementos.

using System;
using System.Collections.Generic;

namespace AplicacionConsola
{
    class Program
    {
        static void Main(string[] args)
        {
            List<string> frutas = new List<string>
            {
                "Manzana",
                "Pera",
                "Uva",
                "Durazno"
            };

            foreach (string fruta in frutas)
            {
                Console.WriteLine(fruta);
            }

            Console.ReadKey();
        }
    }
}

# Ciclo WHILE

LA estructura repetitiva While tiene la particularidad que siempre itera mientras (while) la condicion evaluada sea true, en el momento que sea false esta dejara de iterar, osea primero evaluá la condicion y después ejecutara su código interno.

using System;

namespace AplicacionConsola
{
    class Program
    {
        static void Main(string[] args)
        {
            int x;
            x = 1;
            while (x <= 100)
            {
                Console.Write(x);
                Console.Write(" - ");
                x++;
            }

            Console.ReadKey();
        }
    }
}

# Ciclo DO

La estructura do while es otra estructura repetitiva, la cual ejecuta al menos una vez su bloque repetitivo, a diferencia del while o del for que podían no ejecutar el bloque. Esta estructura repetitiva se utiliza cuando conocemos de antemano que por lo menos una vez se ejecutará el bloque repetitivo. La condición de la estructura está abajo del bloque a repetir, a diferencia del while o del for que está en la parte superior.

using System;

namespace AplicacionConsola
{
    class Program
    {
        static void Main(string[] args)
        {
            int valor;
            string linea;
            do
            {
                Console.Write("Ingrese un valor entre 0 y 999 (0 finaliza):");
                linea = Console.ReadLine();
                valor = Convert.ToInt32(linea);
                if (valor >= 100)
                {
                    Console.WriteLine("Tiene 3 dígitos.");
                }
                else
                {
                    if (valor >= 10)
                    {
                        Console.WriteLine("Tiene 2 dígitos.");
                    }
                    else
                    {
                        Console.WriteLine("Tiene 1 dígito.");
                    }
                }
            } while (valor != 0);

            Console.ReadKey();
        }
    }
}
