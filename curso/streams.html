# Uso de FileStreams

El flujo es la transportación de información en este caso un archivo, este flujo sera por una secuencia de bytes.

# Como crear un archivo

Para ello usaremos la clase FileStream.

//Solo con esta linea creamos un archivo vació
FileStream fs = new FileStream("MiArchivo.txt", FileMode.Create);
1
2

Ahora para agregar datos o texto al archivo usaremos el método Write() y para cerrar el archivo y liberar los recursos usamos seguidamente el método Close().

static void Main(string[] args)
{
    FileStream fs = new FileStream("MiArchivo.txt", FileMode.Create);
    string informacion = "Esta es la información que agregaré a mi archivo";
    fs.Write(ASCIIEncoding.ASCII.GetBytes(informacion), 0, informacion.Length);
    fs.Close();
}

# Como leer un archivo

Para esto tenemos que crear un arreglo de bytes como buffer de almacenamiento, y usaremos el metodo Read(), el cual volcara el stream en el buffer.

static void Main(string[] args)
{
    byte[] getInfo = new byte[100];
    FileStream fs = new FileStream("MiArchivo.txt", FileMode.Open);
    fs.Read(getInfo, 0, (int)fs.Length);
    Console.WriteLine(ASCIIEncoding.ASCII.GetString(getInfo));
    fs.Close();
    Console.ReadKey();
}
