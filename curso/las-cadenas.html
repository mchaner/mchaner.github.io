    # Las Cadenas

    Una cadena es un objeto de tipo String cuyo valor es texto. Internamente, el texto se almacena como una colección secuencial de solo lectura de objetos Char.

    En C#, la palabra clave string es un alias de String. Por lo tanto, String y string son equivalentes y se puede utilizar la convención de nomenclatura que prefiera. La clase String proporciona muchos métodos para crear, manipular y comparar cadenas de forma segura.

    # Declaración de una cadena

    //considerar las comillas dobles para representar la cadena
String cadena = "esto es una cadena";

    # Concatenar

    Para concatenar cadenas String posee un método llamado Concat()

    String nombre = "Pedro";
String apellido = "Fernandez";
String nombreCompleto = nombre + " " + apellido;

Console.WriteLine(nombreCompleto);

nombreCompleto = String.Concat(nombre, apellido);

Console.WriteLine(nombreCompleto);

nombreCompleto = String.Concat(nombre, String.Concat(" ", apellido));

Console.WriteLine(nombreCompleto);

    # Comparar cadenas

    Para poder comparar cadenas tenemos dos opciones usar el método Compare() o el método Equals().

    int comparacion = 0;
string nombre1 = "juan";
string nombre2 = "juan";

//usando compare
comparacion = String.Compare(nombre1, nombre2);

if(comparacion == 0)
{
    Console.WriteLine("Las cadenas son iguales");
}
else
{
    Console.WriteLine("Las cadenas son distintas");
}

//usar equals
if (String.Equals(nombre1,nombre2))
{
    Console.WriteLine("Las cadenas son iguales");
}
else
{
    Console.WriteLine("Las cadenas son distintas");
}

    # Extraer una subcadena

    Aveces es necesario extraer algún valor de una cadena predeterminada y/o formateada, para ello usamos el método Substring().

    String cadena = "El monto de hoy es: 5000";
String resultado = cadena.Substring(20, cadena.Length - 20);

Console.WriteLine(resultado);

    # Remplazar parte de una cadena

    Para algunas reglas de negocio aveces necesitaremos remplazar solo una parte de una cadena por otra, para ello tenemos el método Replace();

    ArrayList fechaPago = new ArrayList();

fechaPago.Add("### Juan Perez");
fechaPago.Add("### Alberto Rozas");
fechaPago.Add("### Luis Garcia");
fechaPago.Add("### Sebastian Almuna");

foreach (string  item in fechaPago)
{               
    String cambio =  item.Replace("###", String.Format("{0:dddd M yyy}", DateTime.Now).ToString());
    Console.WriteLine(cambio);
}
